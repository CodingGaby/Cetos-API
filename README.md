# cetosapi

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```
Install the required packages:

```bash
npm install express csv-parser

```
This project was created using `bun init` in bun v1.1.20. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
